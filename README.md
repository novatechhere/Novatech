# 🚀 NovaTech — Innovation & Venture Building Platform

NovaTech is a structured ecosystem designed to help students, builders, and early-stage founders transform ideas into real-world ventures through validation, execution, and strategic growth.

---

## 🔥 What NovaTech Does

NovaTech is not just a community or a learning platform — it operates as a **venture-building system** combining:

* **Idea Validation**
* **Product Structuring**
* **Execution Support**
* **Growth Strategy**

---

## 🧱 Core Pillars

### 1. Innovation Club

A collaborative environment for:

* Ideation sessions
* Team formation
* Cross-domain collaboration
* Networking with peers and mentors

---

### 2. Problem Lab

A structured execution layer focused on:

* Real-world problem solving
* Sprint-based challenges
* Prototype building
* Expert feedback

---

### 3. Learning Platform

Applied learning system including:

* Project-based tracks
* Peer-led sessions
* Skill development
* Portfolio building

---

## 📊 Audit System (Key Offering)

NovaTech provides structured audit reports for startups and ideas:

### 🥈 Silver

* Idea validation
* Problem identification
* Basic improvements

### 🥇 Gold

* Full product + UX audit
* Growth bottleneck analysis
* Technical recommendations

### 💎 Nova

* Advanced expert analysis
* Growth roadmap
* Marketing & scaling strategy

## 🌐 Website Features

* Multi-step **registration system**
* Input validation & sanitization
* Real-time **data submission via webhook (Make.com)**
* Automated:

  * Google Sheets data storage
  * User confirmation emails
  * Admin notifications

---

## ⚙️ Tech Stack

### Frontend

* HTML5
* CSS3 (Custom Design System)
* Vanilla JavaScript

### Automation & Backend (No-code)

* Make.com (Webhook + Automation)
* Google Sheets (Data Storage)
* Gmail API (Email automation)

---

## 🔄 Data Flow

```text
User Form → Webhook (Make) → 
→ Google Sheets (Store Data)
→ Gmail (User Confirmation)
→ Gmail (Admin Notification)
```
